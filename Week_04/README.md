## 学习笔记：
Dynamic Programming动态规划三步：
		1. 分治：把问题化为子问题
		2. 定义状态空间：通常为一维二维数组 ，最好为一维数组。
		3. Dp方程： 状态转移方程，每一步的状态如何改变，并且适用于所有的子问题。

和递归比较：
	共同点：找到重复子结构。
	不同点：最优子结构，需要有个存储空间来保存实时变化的状态。
